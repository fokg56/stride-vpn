# Stride VPN

<img width="190" height="190" alt="LOGO" src="https://github.com/user-attachments/assets/84192a2a-793f-40f9-94f5-4b239ed3202f" />


Windows VPN-клиент на базе xray-core. TUN и прокси-режим, карта мира, автоматический импорт подписок, тёмная/светлая тема.

## Быстрый старт

1. Скачайте `stride-vpn.exe`
2. Запустите — откроется браузер
3. Импортируйте подписку или вставьте vless:// ссылку
4. Нажмите на сервер и подключитесь

При первом подключении через TUN система запросит права администратора (UAC). Достаточно один раз за сессию — при переподключении окно не появится.

## Как работает

Клиент запускает веб-интерфейс на локальном сервере (по умолчанию `localhost:8080`). В браузере отображается список серверов, статус подключения, карта мира и пинг.

**Режимы подключения:**
- **TUN** — виртуальный адаптер, через который идёт весь трафик системы. Требует админ-права.
- **Прокси** — SOCKS5 + HTTP на `127.0.0.1:10808`. Не требует прав (системный прокси устанавливается автоматически, можно отключить в настройках).

Поддерживаются все транспорты: WebSocket, XHTTP (splithttp), gRPC, TCP, QUIC, KCP, HTTP/2. REALITY-подключения работают через utls.


## Сборка из исходников

```
git clone https://github.com/fokg_56/stride-vpn.git
cd stride-vpn
go build -o stride-vpn.exe -ldflags="-s -w" ./cmd/
```

Требуется Go 1.21+, xray-core (подтягивается как зависимость). Внешних зависимостей больше нет — всё необходимое вшито в бинарник (wintun.dll, иконки, HTML/JS/CSS интерфейс).

## Почему я это сделал

Существующие Windows-клиенты для xray-core перегружены — в них куча кнопок, графиков, настроек, которых обычный пользователь не понимает. Мне хотелось сделать что-то простое: скачал, вставил ссылку, подключился. Без лишних движений.

Исходники открыты — если что-то не нравится, правите сами.
